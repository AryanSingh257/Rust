# Guessing Game in Rust
You have to find the random number between 1 to 100 by guessing multiple times with the help of the hints given by the code.

# Tree
This shows the structure of the dir:
```
 .
 ├── Cargo.lock
 ├── Cargo.toml
 ├── README.md
 └── src
             └── main.rs
```

# Installation and run

You can run this program by running the below cmd:

```
git clone https://github.com/AryanSingh257/Rust
cd guess_game
cargo run
```

Ensure you have cargo(comes with rust) downloaded. Follow official **Rust** docs for installation guide.
